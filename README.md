# Lukas Schmelzeisen's dotfiles

My system configuration.

## Installation

[Install chezmoi](https://www.chezmoi.io/docs/install/) and run:

```sh
chezmoi init --apply --verbose https://github.com/lschmelzeisen/dotfiles
```
